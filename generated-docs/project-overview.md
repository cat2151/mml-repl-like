# Project Overview

## プロジェクト概要
- MML（Music Macro Language）をリアルタイムで編集・試聴できるREPL（Read-Eval-Print Loop）ライクなWebアプリケーションです。
- MMLの作成、フレーズやコード進行の試作において、即座に音を確認しフィードバックサイクルを高速化することを目的としています。
- 入力文字ごとやカーソル移動時に音が鳴る機能、選択範囲や現在行の即時演奏、トランスポーズ機能などを提供します。

## 技術スタック
- フロントエンド: HTML5, CSS3, JavaScript (Webアプリケーションの構築基盤として使用されており、インタラクティブなMMLエディタを実現しています。)
- 音楽・オーディオ: 
    - Sionic.js (Fork版): SiONのJavaScript移植版であり、オーディオシンセシス機能を提供し、MMLのリアルタイム演奏を可能にしています。
    - SiON: 高品質なサウンド生成を行うためのソフトウェアシンセサイザー技術であり、Sionic.jsの基盤となっています。
    - pico.js: 軽量なJavaScriptライブラリで、オーディオ処理やその他のユーティリティ機能を提供していると推測されます。
