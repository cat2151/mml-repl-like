# Project Overview

## プロジェクト概要
- MML（Music Macro Language）をリアルタイムで記述し、即座にその音を演奏できるWebアプリケーションです。
- 入力文字ごとやカーソル移動、特定のショートカットキーで、MMLのフレーズやコード進行を手軽に確認・編集できます。
- 音楽制作におけるMMLの試行錯誤サイクルを加速させるための開発支援ツールです。

## 技術スタック
- フロントエンド:
  - HTML: Webアプリケーションの構造を記述するためのマークアップ言語です。MMLの入力インターフェースや再生コントロールなどのUIを提供します。
  - JavaScript: Webアプリケーションの動的な挙動を実装するためのプログラミング言語です。MMLの解析、音源の制御、ユーザーインタラクションの処理を行います。
- 音楽・オーディオ:
  - SiON: ソフトウェアシンセサイザーのライブラリです。Webブラウザ上でMMLに基づいて音源を生成する中核技術として利用されています。
  - Sionic.js (Fork版): SiONをJavaScriptから扱いやすくするためのラッパーライブラリです。本プロジェクトでは、開発者がカスタマイズしたフォーク版が使用されています。
  - MML (Music Macro Language): 音楽をテキストで記述するための記法です。本プロジェクトの中心となる入力言語として利用されます。
  - pico.js: 軽量なJavaScriptライブラリです。プロジェクトの機能の一部をサポートするために利用されています。

- 言語機能: なし
- 自動化・CI/CD: なし
- 開発標準: なし
