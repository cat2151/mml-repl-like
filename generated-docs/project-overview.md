# Project Overview

## プロジェクト概要
- MML（Music Macro Language）をリアルタイムで編集し、インタラクティブに音を再生できるウェブアプリケーションです。
- キーボード入力やカーソル移動に連動して即座にMMLの音を鳴らし、音楽制作の試行錯誤サイクルを高速化します。
- フレーズ作成、コード進行の試行、転調などのMMLベースの音楽制作を直感的に支援するツールです。
